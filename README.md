# App Center

A simple GUI tool to install essential Windows applications instantly using `winget`.

## Usage
1. Download `AppCenter.exe` and `apps.json` from the [Releases](https://github.com/mackka2k/app-center/releases) page.
2. Keep them in the same folder.
3. Run `AppCenter.exe`.
4. Select apps and click **Install**.

## Requirements
- Windows 10/11
- Internet connection
